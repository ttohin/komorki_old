## Building (currenly only for macosx Sierra)

Download cocos2dx dependencies

```bash
cd komorki/cocos2d
python download-deps.py
```

Open xcode workspace

```bash
open proj.ios_mac/Komorki.xcodeproj
```
In xcode select target **Komorki**, run and enjoy!
